OTTO Modal Windows Test

Test all the modal windows to ensure they open and close properly:

Modal Windows to Test:

1. Settings Modal

  • Button: Gear icon (Row 1, Area 1)
  • Should show: Application, MIDI, and Audio settings
  • Contains: Factory reset button for Default preset

2. Link Modal

  • Button: Link icon (Row 1, Area 1)
  • Should show: Link configuration placeholder
  • Purpose: Will control parameter linking between players

3. Mixer Modal

  • Button: Faders icon (Row 3, Area 4)
  • Should show: "Kit Mixer - Player X" (X = current player number)
  • Purpose: Will control individual drum channel levels

4. Drum Kit Edit Modal

  • Button: Pencil icon (Row 3, Area 2)
  • Should show: "Edit Drum Kit - Player X" (X = current player number)
  • Purpose: Will allow editing drum kit sounds and mappings

5. Favorites Edit Modal

  • Button: Pencil icon (Row 4, Area 1)
  • Should show: "Edit Pattern Group - [Group Name]"
  • Purpose: Will allow editing pattern groups (16 patterns per group)

6. Preset Management Modal

  • Button: Pencil icon (Row 1, next to preset dropdown)
  • Should show: List of presets with management options
  • Contains: New, Undo, Reset Default buttons

Test Steps for Each Modal:

  1. Click the button to open modal
  2. Verify modal appears with correct title
  3. Check that background is dimmed
  4. Click the X button to close
  5. Reopen and click outside modal to close
  6. Verify ESC key closes modal (if implemented)

Console Commands for Testing:

// Manually open modals
otto.openSettingsModal()    // Note: this uses onSettingsClicked()
otto.openLinkModal()
otto.openMixerModal()
otto.openKitEditModal()
otto.openFavoritesModal()
otto.openPresetModal()

// Check current player (affects mixer/kit edit titles)
otto.currentPlayer

// Switch players and test modals update
otto.switchToPlayer(2)
otto.openMixerModal()  // Should show "Player 2"

  • ✅ All modals open when their buttons are clicked

  • ✅ X button closes modal

  • ✅ Only one modal visible at a time
  • ✅ Background is properly dimmed
  • ✅ Modals are centered on screen
